    Welcome to the documentation of EVAdb. The Exome Variant Analysis Database serves to streamline analysis of exome cases for the users. In this project we aim to provide a concise documentation for users and administrators of the software alike.

    Repositories

    All of EVAdb's source code and files necessary to run the application, can be found on github:

    • EVAdb
    • Pipeline

    EVAdb is in active use in Munich and Bonn. It has been developed by Tim Strom in Munich and has been used to analyze more than ten thousand exome case over the last years. It is based on perl scripts and a MySQL database. It can be installed directly on bare-matel or a VM or build and deployed via docker.
